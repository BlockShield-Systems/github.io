Technical Questions

You receive two versions of each image:

  • High-Resolution: PNG or JPG (2048-4096px) - For print, large displays
  • Web-Optimized: Compressed JPG (1200-1920px) - For fast website loading

Custom sizes and formats (WebP, AVIF, etc.) available upon request.

Standard deliverables:

  • High-res: 2048px - 4096px (width or height depending on orientation)
  • Resolution: 300 DPI (print-quality)
  • Web: 1200px - 1920px (optimized file size)

Common aspect ratios available:

  • 16:9 (website banners)
  • 1:1 (social media)
  • 9:16 (mobile/stories)
  • 4:3 (standard)
  • Custom ratios upon request

We work with RGB color space (digital), so we convert Pantone to RGB. The result is typically 90-95% accurate to the Pantone swatch.

For best results, provide:

  • Hex codes (#667eea)
  • RGB values (102, 126, 234)
  • Or Pantone codes (we'll convert)
